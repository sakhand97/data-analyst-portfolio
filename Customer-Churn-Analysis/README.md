## Customer Churn & Behavior Analysis

### Business Objective
Understand customer churn behavior and identify high-value customer segments.

### Dataset
- IBM Telco Customer Churn Dataset (public dataset)

### Tools Used
- SQL
- Python (Pandas, NumPy, Matplotlib)

### Key Analysis
- Churn rate analysis
- Customer segmentation by tenure and revenue
- Behavioral patterns affecting churn

### Outcome
Provided insights on churn drivers and high-value customers to support retention strategies.
